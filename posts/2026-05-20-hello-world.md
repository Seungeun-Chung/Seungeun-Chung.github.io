안녕하세요! 포트폴리오 블로그를 시작합니다.

이 블로그에서는 개발 관련 지식, 프로젝트 후기, 학습 기록 등을 공유할 예정입니다.

## 포스트 작성 방법

새 포스트를 추가하려면 두 가지만 하면 됩니다.

1. `posts/` 폴더에 `.md` 파일 추가
2. `posts/index.json`에 메타데이터 등록

```json
{
  "file": "2026-06-01-new-post.md",
  "title": "새 포스트 제목",
  "description": "포스트 요약",
  "date": "2026-06-01",
  "tags": ["태그1", "태그2"],
  "category": "카테고리"
}
```

## 코드 예시

```python
def hello():
    print("블로그에 오신 것을 환영합니다!")

hello()
```
